# Agent 技能清单 (Skills)

## 🚀 技能列表

### 1. 🎬 基于技术文档/论文的视频稿创作 (Technical Paper/Article Video Script Generation)
**描述：** 基于技术文档/论文/博客文章的核心数据（如 Abstract、核心公式、实验结果），创作 10 分钟口播视频稿。包括脚本、画面描述、制作指南。
**能力：** 深度阅读技术文档、提取核心信息、转化为易于理解的内容、生成完整的视频时间线和画面描述。
**输入：** 论文/文档链接或核心发现（如 "极简配方"）、目标受众、视频时长。
**输出：** Markdown 格式的视频稿文件（包含时间轴、画面、旁白、制作指南）。
**触发条件：** 当用户要求“根据某篇论文/技术文档创作视频稿”或“基于某个技术概念创作视频”时。

### 2. 🔍 浏览器自动化与搜索 (Browser Automation & Search)
**描述：** 使用 Browser 工具进行网页导航、快照、点击操作，并使用 Web Fetch 获取网页内容，进行信息检索和交叉验证。
**能力：**
- 导航到指定 URL（使用 `navigate` action）。
- 获取页面快照（使用 `snapshot` action）。
- 执行页面交互（使用 `act` action with `kind: "click"`）。
- 获取网页全文（使用 `web_fetch` tool）。
- 信息提取与交叉验证（多源对比）。
**触发条件：** 当用户要求“搜索某信息”、“打开某网页”、“验证某内容”时。

### 3. 📂 文件与 Git 管理 (File & Git Management)
**描述：** 读取、创建、更新文件，执行 Git 命令。
**能力：**
- 使用 `read` 工具查看文件内容。
- 使用 `write` 工具保存更改。
- 使用 `exec` 工具运行 `git add`、`git commit`、`git push`。
- 管理本地文件系统和远程代码仓库。
**触发条件：** 当用户要求“保存文件”、“提交代码”、“推送到 GitHub”时。

### 4. 🧠 内容整合与深度分析 (Content Integration & Deep Analysis)
**描述：** 深度阅读技术文档或论文，提取核心信息，并将其转化为易于理解的内容。
**能力：**
- 使用 `web_fetch` 获取论文全文。
- 分析 Abstract、核心公式、实验结果。
- 将技术概念转化为通俗语言。
- 结合用户需求（如“视频稿”）进行创作。
**触发条件：** 当用户要求“拉取论文”、“详读某技术”或“基于某技术创作内容”时。
